# To-do App

## A simple to-do app built with golang learning the fundamentals.

## About the project
This project was built following the tuturial of tech with Nana.
I implemented what I learned using slices, functions, variables and simple flow.

## Features
- Printing the tasks.
- List all the tasks.
- add a new task
- update the task list

  ## How to run the project
bash:  git clone https://github.com/AhmedGoda1/Practicing-Go-To-do-app-
